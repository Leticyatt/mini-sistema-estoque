O **Controle de Estoque** é uma aplicação web completa (SaaS) desenvolvida para facilitar o controle e a gestão de inventário de pequenas e médias empresas. Ele oferece uma interface moderna, intuitiva e responsiva, permitindo que os gestores acompanhem a entrada e saída de produtos e recebam alertas de estoque em tempo real.

---

## Funcionalidades:

- **🔒 Autenticação de Usuários:** Sistema seguro de Login, Cadastro e Recuperação de Senha utilizando o Supabase Auth.
- **📊 Dashboard Interativo:** Visão geral do estoque atualizado em tempo real.
- **📝 Gestão de Produtos:** Cadastro, edição, listagem e exclusão de itens (CRUD completo).
- **🔄 Movimentação de Estoque:** Registro rápido de entradas e saídas de mercadorias.
- **⚠️ Alertas Inteligentes:** Painel de avisos automáticos para produtos com estoque baixo.
- **📱 Design Responsivo:** Interface adaptada para funcionar perfeitamente em computadores, tablets e smartphones.

---

## Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias e bibliotecas:

**Front-end:**
- [React](https://reactjs.org/) + [Vite](https://vitejs.dev/) - Estrutura principal e empacotador.
- [Tailwind CSS](https://tailwindcss.com/) - Estilização moderna e utilitária.
- [Lucide React](https://lucide.dev/) - Biblioteca de ícones.
- [React Router Dom](https://reactrouter.com/) - Navegação dinâmica entre telas.

**Back-end & Infraestrutura:**
- [Supabase](https://supabase.com/) - Banco de Dados PostgreSQL e serviço de Autenticação.
- [Vercel](https://vercel.com/) - Hospedagem e CI/CD.
